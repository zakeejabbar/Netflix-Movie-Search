"Netflix Movie Search"


This program takes in data on movies, stores them in a binary search tree with movie numbers as keys. The user then enters a number they would like to search for and program finds the movie title if it is in the tree otherwise it says movie not found.


To run this program:
gcc main.c -o myProg
./myProg

Sample User Input:

movies.csv (other movie files in Data Files Folder)
123 (Some movie #)
